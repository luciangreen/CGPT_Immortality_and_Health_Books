# CGPT Immortality and Health Books

Use these books in <a href="https://github.com/luciangreen/Time_Machine/">Time Machine</a> to help people become immortal or have good health with time crystals.

# Prerequisites

* Please download and install SWI-Prolog for your machine at `https://www.swi-prolog.org/build/`.

# 1. Install manually

Download <a href="http://github.com/luciangreen/CGPT_Immortality_and_Health_Books/">this repository</a> and the repositories these repositories depend on.

# 2. Or Install from List Prolog Package Manager (LPPM)

* Download the <a href="https://github.com/luciangreen/List-Prolog-Package-Manager">LPPM Repository</a>:

```
mkdir GitHub
cd GitHub/
git clone https://github.com/luciangreen/List-Prolog-Package-Manager.git
cd List-Prolog-Package-Manager
swipl
['lppm'].
lppm_install("luciangreen","CGPT_Immortality_and_Health_Books").
halt
```
